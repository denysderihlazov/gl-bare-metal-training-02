\# Bare Metal Training 02: LEDs, Buttons, Interrupts



STM32F407 project implementing LED blinking schemes controlled by buttons using EXTI interrupts.



\## Features



\* 3 blinking schemes

\* Previous / Next scheme selection

\* Speed up / Slow down blinking

\* ON/OFF blinking control

\* EXTI interrupt handling for all buttons



\## Blinking Schemes



1\. All LEDs blink together

2\. Running LED sequence

3\. Alternating LED pairs
